FiloShare Release Notes (Windows)

Version 1.1

Important Installation Notes:

  • Recommended Installation Location: It is highly recommended to install FiloShare in the default location during the installation process.
  • Avoid Installation in Program Files Directory: Please refrain from installing the software within the Program Files directory in the C drive, as it may lead to potential issues.

Installation and Setup Instructions:

  1. Download Software:
    • Download the installer executable file (filoshare-setup.exe) from the provided source.
  2. Installation Process:
    • Double-click on the downloaded executable file to initiate the installation process.
    • Follow the instructions provided by the installation wizard to complete the installation successfully.
  3. Launching the Software:
    • Upon successful installation, you can launch FiloShare by:
      • Clicking on the desktop shortcut (if created during installation).
      • Searching for "FiloShare" in the installed applications menu and launching it from there.
  4. Accessing the Application:
    • Once FiloShare is launched, access the application by opening a web browser and entering the following URL: http://localhost:5000.
    • On the homepage, you will find a shareable link that can be used to access the application from other devices for file sharing purposes.

Note Regarding Network Setup:

  • Single Device Requirement: In a network setup, only one device needs to run the FiloShare software.
  • File Sharing Across Devices: Other devices within the same network can share files seamlessly by connecting to the shareable link provided on the device running the FiloShare software.
