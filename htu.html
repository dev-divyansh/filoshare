How to Use FiloShare

Step 1: Run the FiloShare Desktop App

The first step is to run the FiloShare desktop app on your computer.

Step 2: Obtain Connection URL

Once the app is running, you will see a connection URL displayed on the home page. This URL should be shared with other devices you want to connect with.

Open the URL in the web browser of all other devices you want to share files with, such as your phone or other computers. Make sure that all devices are connected to the same network. Devices connected via a hotspot are also considered on the same network.

Step 3: Share or Download Files

Once all devices are connected, you can start sharing files. You can also download files shared by others. When you are done, simply close the web browser.
